# puzzled
Brandwatch 2019 puzzler: Crossword grid generator

## Install

```
make
```

### Run

```
./puzzled -M <M> -N <N> [-d <dictionnary>]
```


### Test your grid

```
./puzzled -t <test_file_path>
```

The validation uses this format
```
abacay......
......abacus
ac..inetan..
..ne....ther
```
